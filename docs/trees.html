    • cells.cell._CellException

      • cells.cell.InputCellRunError: An attempt to run() an InputCell was made.
      • cells.cell.RuleCellSetError: RuleCells may not be set.
      • cells.cell.SetDuringNotificationError: An attempt at a non-deferred set() happened during propogation.
      • cells.cell.RuleAndValueInitError: Both rule and value were passed to __init__.

    • cells.observer.Observer: Wrapper for a function which fires when a Model updates and certain conditions are met.
    • type: type(object) -> the object's type type(name, bases, dict) -> a new type
      • cells.model.ModelMetatype
    • cells.observer.ObserverAttr: Wrapper for Observers within Models.
    • cells.model.Model: A class in which CellAttrs may be used.
      • cells.family.Family: Family
    • cells.cellattr.CellAttr: CellAttr
    • cells.cell.Cell: The base Cell class.
      • cells.cell.RuleCell: A cell whose value is determined by a function (a rule).
        • cells.cell.LazyCell: A RuleCell which does not update upon propogation.
          • cells.cell.AlwaysLazyCell: A LazyCell variant that is *always* Lazy
          • cells.cell.OnceAskedLazyCell
          • cells.cell.UntilAskedLazyCell: A LazyCell who converts to a normal RuleCell after its first post-init get()

      • cells.cell.RuleThenInputCell: Runs the rule to determine initial value, then acts like a InputCell
      • cells.synapse.Synapse: A very specialized Cell variant.
