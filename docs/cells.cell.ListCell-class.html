Class ListCell
source code

                ListCell
A input cell whose value is initialized to []. An ordinary InputCell doesn't act like we'd like it to in this case: 
>>> import cells
>>> class A(cells.Model):
...     x = cells.makecell(value=[])
...     @cells.fun2cell()
...     def xlen(self, prev):
...         return len(self.x)
... 
>>> a = A()
>>> a.x
[]

>>> a.xlen
0

>>> a.x.append("foo")
>>> a.x
['foo']

>>> a.xlen
0
But if we specify a ListCell, it should work as we expect it: 
>>> import cells
>>> class A(cells.Model):
...     x = cells.makecell(value=[])
...     @cells.fun2cell()
...     def xlen(self, prev):
...         return len(self.x)
... 
>>> a = A()
>>> a.x
[]

>>> a.xlen
0

>>> a.x.append("foo")
>>> a.x
['foo']

>>> a.xlen
1
Note that unchanged_if acts on list elements rather than the entire value.
