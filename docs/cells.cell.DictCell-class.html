                    DictCell
A input cell whose value is initialized to {}. An ordinary InputCell doesn't act like we'd like it to in this case: 
>>> class A(cells.Model):
...     x = cells.makecell(value={})
...     @cells.fun2cell()
...     def xkeys(self, prev):
...         return self.x.keys()
... 
>>> a = A()
>>> a.x
{}

>>> a.xkeys
[]

>>> a.x['foo'] = 'bar'
>>> a.x
{'foo': 'bar'}

>>> a.xkeys
[]
But if we use a DictCell, this will act like we'd like it to: 
>>> class A(cells.Model):
...     x = cells.makecell(value={}, type=DictCell)
...     @cells.fun2cell()
...     def xkeys(self, prev):
...         return self.x.keys()
... 
>>> a = A()
>>> a.x
{}

>>> a.xkeys
[]

>>> a.x['foo'] = 'bar'
>>> a.x
{'foo': 'bar'}

>>> a.xkeys
['foo']
