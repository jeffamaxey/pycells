Class Observer
source code

object --+
         |
        Observer
Wrapper for a function which fires when a Model updates and certain conditions are met. Observers may be bound to specific attributes or whether a function returns true when handed a cell's old value or new value, or any combination of the above. An observer that has no conditions on its running runs whenever the Model updates. Observers run at most once per datapulse.
